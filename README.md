# dnd combat sim
### by Lois Overvoorde and Oliver Gould

## Prerequisites
The code requires a c++ compiler, assumed to be g++, and make.

## Compiling
To compile, use the following in the command line:

	make

## Running
To run the code, use the following in the command line:

    ./dndcs
